# TiCEX - Tinkers' Construct EX

🌐 他の言語: [English](README.md)

様々なmodの拡張要素を追加する、Tinkers' Constructのアドオン。
主に各modの**最強装備**を踏襲したパーツや改造を提供する。星の力で、オリジナルの究極装備を組み立てよう！

## 前提mod

- [Tinkers' Construct](https://modrinth.com/mod/tinkers-construct)
- [AddonAPI v1.2.0](https://modrinth.com/mod/addonapi)

## 対応バージョン
- forge mc1.20.1

## 連携mod
- Re-Avaritia
- Mekanism
- Draconic Evolution
- SlashBlade:Resharped
- Timeless and Classics Zero
- Apotheosis
- Create (v6のみ、v5でも起動しますが連携しません)
- Iron's Spells n' SpellBook(他のTiCアドオンと併用できます)
- Project E
- CC: Tweaked(WIP)
- Curios
- Tinkers' Things
- Sakura Tinker
