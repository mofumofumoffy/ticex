# TiCEX - Tinkers' Construct EX

🌐 他の言語: [English](README.md)

様々なmodの拡張要素を追加する、Tinkers' Constructのアドオン。
主に各modの**最強装備**を踏襲したパーツや改造を提供する。星の力で、オリジナルの究極装備を組み立てよう！

## 前提mod
### 全てのバージョン
[Tinkers' Construct](https://www.curseforge.com/minecraft/mc-mods/tinkers-construct)

### その他
|version|mods|
|--|--|
|mc1.20.1|AddonAPI v1.1.0(リンク準備中)|

## 対応バージョン
- forge mc1.20.1

## 連携要素
- Avaritia(Re-Avaritia-forged推奨)
 - Infinity マテリアル
 - Neutronium マテリアル
 - Crystal Matrix マテリアル
 - クリエイティブ飛行が可能になるCelestial モディファイア
 - 煌めく銀河のシェーダー
- Mekanism
 - 放射線から身を守るRadiation Shielding モディファイア
 - MekaPlate アーマー: MekaSuit モジュールをサポートするプレートアーマー
- Draconic Evolution
 - Draconic Evolution モジュールをサポートする Evolved モディファイア
 - 渦巻く混沌のシェーダー

## その他の追加要素
- 星の力
- いくつかの新合金
- FEに応じて温度が上がる乾式製錬炉、鋳造炉の電気ユニット
