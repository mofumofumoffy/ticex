# 素材特性
## TiCEX

### エーテル
武器・ツールに使用できる素材です。
- 偏向:本来攻撃の当たらないモブにもダメージが与えられます。
- キレッキレ:ツールが必ずクリティカルヒットします。

### セラムカタリスト
- 再誕:素材の特性を引き継ぎます。

## Re:Avaritia

### クリスタルマトリックス:
武器・ツールに使用できる素材です。
- アフターショック:チャージ攻撃時、追加で魔法ダメージを与えます。
- 貪欲

### ニュートロニウム
防具に使用できる素材です。
- 超高密度:攻撃を受けたとき、ニュートロン塊を生成することがあります。
- 重装:ノックバックを抑えます。

### インフィニティ
武器・ツール・防具に使用できる素材です。
- 全能:すべてのブロックを一撃で破壊し、すべてのモブを一撃でキルします。
- 幸運X:ツールによるドロップや経験値にボーナスを与えます。
- 不可壊:ツールが壊れることがなくなります。
- 岩盤破壊:ツールが岩盤を破壊可能になります。
- 超越:装備部位ごとに被ダメージを25%軽減します。

## Draconic Evolution
すべて近接武器・ツールにのみ使用できる素材です。<br>
ドラコニウム、ワイバーン、ドラコニック、カオティックすべてで共通の特性を持ちます。
- 魂狩り:ツールでモブの魂をドロップする可能性があります。
- 裁断

# 改造

## TiCEX
- エンボス:任意のパーツの特性を付与します（一度のみ）

## Re:Avaritia
- 天翔:クリエイティブ飛行が可能になります。
- エンデストショット:耐久を消費してエンデストパールの効果があるエンデストショットを装填します。

## Mekanism
- 放射線防護:Mekanismの防護装備と同等の効果を与えます。MekaPlateには適用できません。
- Mekanic:防具をMekaPlateに変換します。

## Draconic Evolution
- エボルト:Draconic Evolutionのモジュールを使用できるようになります。モジュールのティアはレベルによって異なります。防具には適用できません。前提として補強V、ネザライトが必要です。
- インジェクション:アビリティとアップグレードスロットを追加します。

## SlashBlade:Resharped
- 魂魄:抜刀剣に付与可能な一部のエンチャントをエンチャントの本から継承できます。

| エンチャント | 最大レベル |
| ---- | ---- |
| 火炎耐性| 4 |
| 落下耐性| 4 |
| 水中呼吸| 3 |
| 射撃ダメージ増加| 5 |
| パンチ| 2 |
| 茨の鎧| 3 |
| ソウルスピード| 3 |

- 拵:コアに使用した抜刀剣のエンチャント以外のステータスを引き継ぎます。妖刀化しているものにのみ適用できます。
- 秘伝:ProudSoul増加、SAの変更、特殊効果の付与ができます。掛け台を用いた強化の代わりとなります。

## Apotheosis
- オーバーロード:ツールにエンチャントをエンチャントの本から継承できます。互換性のないエンチャントは継承できません。
- オーバーライド:限界を突破したエンチャントを継承できます。

## Create（v6のみ）
- ダンボール:すべての防具に適用されているとき、ダンボールに隠れることができます。

## Iron's Spells n' SpellBook
- オーバー詠唱:消費マナコストが上昇しますが、スペルに特性の効果を乗せることができます。

## Curios
- 桁違い:ツールやアーマーが弱体化する代わりに、専用のCuriosスロットに装備できるようになります。敵対的インベントリ操作に疑似的な耐性を得られます。

## Project E
ジェム装備の特性を防具に付与します。
- 深淵
- 煉獄
- 重力
- 台風

## Psi
- サイオン特性:ツールがサイメタル装備のように特定条件で魔法を発動させることができます。スペルスロット+1
- センサー:使用したセンサーに応じた条件で魔法が発動するようになります。頭装備にのみ適用できます。

## CC:Tweaked
- モデム:装備する人の行動に応じてポケットコンピューターにいくつかのイベントを渡します。

## Integrations with Other TiC Addons
### Tinkers' Things
- 多層: Embosses Seram Gear with missing materials to simulate Laminar Armor properties.

### Sakura Tinker
- 花吹雪: Applies custom TiCEX modifiers to conflicting materials. Also, shaders are automatically applied to infinity materials by default.
